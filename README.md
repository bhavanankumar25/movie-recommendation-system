# Movie Recommendation System

## Overview
This project is a Machine Learning-based recommendation system that suggests movies based on similarity.

## Features
- Recommend movies based on input
- Uses TF-IDF and cosine similarity
- Fast and simple recommendation engine

##  How It Works
This project uses a **Content-Based Filtering** approach:

- **TF-IDF (Term Frequency - Inverse Document Frequency)**  
  Converts movie genres into numerical vectors  

- **Cosine Similarity**  
  Measures similarity between movies  

Movies with higher similarity scores are recommended to the user.


## Technologies
- Python
- Pandas
- Scikit-learn

## How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Open the notebook:
   movie-recommendation-system.ipynb

3. Run all cells step by step

4. Enter a movie name when prompted
